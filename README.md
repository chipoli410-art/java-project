♻️ 리사이클(Recycle) JAVA 프로젝트 - 분리수거 안내 서비스

🎯 프로젝트 소개 및 목적 본 프로젝트는 Java와 MySQL 데이터베이스를 기반으로 하여, 다음과 같은 기능을 제공함으로써

사용자의 올바른 분리수거 습관 형성을 돕는 SW이다.

기존 정보 제공 방식의 한계와 동기부여 부족 단순한 텍스트 나열식 정보 제공은 사용자의 흥미를 끌지 못하며, 자발적인 학습을 유도하기 어렵다.

따라서 사용자가 분리배출 정보를 쉽고 재미있게 습득하고, 지속적으로 실천할 수 있도록 돕는 요소를 도입한 소프트웨어가 필요하다 생각하여 본 프로젝트를 만들기로 결심하였다.

  1. 참여형 학습 (Quiz): 퀴즈를 통해 분리배출 지식을 자연스럽게 습득하게 하고,'1일 1회 참여 제한' 기능을 DB로 구현하여 꾸준한 접속을 유도한다.
  
  2.분리수거 품목 포인트화(Recycling separation):  사용자가 기입한  분리수거 품목별로 포인트화 하여 포인트를 합쳐 계산하여 사용자에게 지금동안 모은 포인트를 볼수 있다.
  
  3.보상 및 경쟁 심리 (Point & Ranking): 분리수거 실천과 퀴즈 정답 시 포인트를 지급하고,실시간 Top 5 랭킹을 시각화하여 사용자의 성취감과 경쟁 심리를 자극한다.
  
  4.데이터 기반 관리 (MySQL DB): 초기 기획의 단순 파일 입출력(TXT) 방식에서 벗어나,MySQL을 도입하여 회원 정보, 포인트 적립/사용 내역, 퀴즈 기록 등을 체계적이고 무결하게 관리한다.
  
  5.순환 구조의 보상 체계 (Store): 획득한 포인트를 사용하여 상품을 구매하는(가상 시뮬레이션) 기능을 구현함으로써, '실천 -> 보상 -> 사용'으로 이어지는 자원 순환의 가치를 소프트웨어 내에서 경험시킨다.
💻 코드 설치 및 실행 방법

1.조건(전제조건)

1.Java 개발 환경 (JDK) : Java 11 이상 버전 설치

2.MySQL Server : MySQL 8.0+ 버전 설치 및 실행

3.개발툴 : Eclipse IDE 설치

4.MySQL JDBC Driver : mysql-connector-j-x.x.x.jar파일 다운로드
프로젝트 다운로드 및 설정

 1. 프로젝트 대상: git clone [깃허브 저장소 주소 URL]
    
 3. Eclipse 프로젝트 임포트:
    
    Eclipse 실행 -> File-> Import-> General-> Existing Projects into Workspace선택
    
    Select root directory에 덤프한 프로젝트 폴더를 지지하고 임포트합니다.
데이터베이스 설정

  1.MySQL 접속: MySQL Workbench 또는 콘솔을 사용하여 MySQL 서버에 접속합니다.
  
  2.데이터베이스 생성: 프로젝트에서 사용 데이터베이스를 생성합니다.
  
  3.생성: 프로젝트를 시작하여 DB 테이블 파일을 생성합니다.
JDBC 드라이버 및 DB 연결 설정

  1.드라이버 추가:
  
    프로젝트 폴더에서 우클릭 -> Properties-> Java Build Path-> Libraries탭 선택
  
    Add External JARs...다운로드한 mysql-connector-j-x.x.x.jar파일을 추가합니다.

  2.DB 접속 정보 수정:
  
   프로젝트 db.RecycleDB.java파일을 열어 MySQL 인터페이스 정보
  
   (URL, USER, PASSWORD)를 실제 사용자 환경에 맞게 수정합니다.
5.실행

  1.프로젝트의 메인 클래스(외계로 Main.java또는 App.java등)를 찾고 있습니다.
  
  2.메인 클래스에서 우클릭 후 Run As-> Java Application을 선택하여 프로그램을 실행합니다.
